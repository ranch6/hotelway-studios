Every photo & video on the site, in one place.

This board previews exactly what assets/media.js is pointing at. Online it sits behind the studio password (middleware.js + /api/login); opened locally it just works.

How to change any image (or add a video)

  1. Drop the new photo or video into the assets/ folder.
  2. Open assets/media.js in any text editor — every slot below has a matching entry there, in the same order, under the same section name.
  3. Point the entry's src: at your new filename (for videos, add video: true and an optional poster: still).
  4. Save, refresh this page to check it, then commit & push — Vercel publishes automatically.

Blank templates — copy into assets/media.js

A new photo tile, a new video/reel tile, and a whole new portfolio series:

// photo tile
{ src: "assets/YOUR-PHOTO.jpg", alt: "describe the photo",
  title: "Tile Title", meta: "Small · Uppercase · Caption" },

// video / reel tile (plays inline, muted, looping)
{ src: "assets/YOUR-REEL.mp4", video: true, poster: "assets/OPTIONAL-STILL.jpg",
  title: "In Motion", meta: "Arrival · 15s" },

// a whole new portfolio series (goes in the portfolio: [...] list)
{
  eyebrow: "Series 04",
  heading: "Your new series title.",
  lede: "One quiet sentence about what this series is.",
  items: [
    // photo/video tiles here
  ],
},
